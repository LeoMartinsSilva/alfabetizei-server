# Como Rodar o projeto
****

## Passo 1: Instalar o IntelliJ
Recomenda-se utilizar a IDE IntelliJ para rodar o projeto, pois ela facilita a configuração e execução do ambiente de desenvolvimento.

## Passo 2: importar o projeto no Intellij
Abra o IntelliJ e importe a pasta `alfabetizei-server`.
Após a importação, aguarde até que todas as dependências sejam baixadas automaticamente.

## Passo 3: Selecionar a versão do Java
Certifique-se de que o projeto está configurado para usar a **versão 17 do Java**, garantindo assim a compatibilidade e o correto funcionamento da aplicação.

## Passo 4: Adicionar o plugin de Lombok no intelliJ
O projeto utiliza a biblioteca **Lombok** para geração automática de getters, setters, construtores, entre outros.
Por isso, é necessário instalar o plugin Lombok no IntelliJ.
Mais informações podem ser encontradas na documentação oficial:
https://projectlombok.org/setup/intellij

## Passo 5: Iniciar o projeto
Navegue até o arquivo `AlfabetizeiServerApplication.java`, localizado em `br/app/alfabetizei`, e clique no botão verde de “Play” exibido no topo da IDE para iniciar a aplicação.
